Em React, você pode passar propriedades (props) para componentes para que eles possam usar esses valores dentro de si. 
Aqui está como você pode fazer isso em componentes de classe e funcionais:


Para enviar e acessar props (propriedades) em React, siga estes passos:

Enviando Props
Props são passadas de um componente pai para um componente filho como atributos de um elemento JSX.

Exemplo

import React from 'react';


function Saudacao(props) {
  return <h1>Olá, {props.nome}</h1>;
}

//ENVIANDO
function App() {
  return <Saudacao nome="João" />;
}

export default App;

Neste exemplo, a prop nome é passada do componente App para o componente Saudacao.

Acessando Props
No componente filho, você acessa as props através do objeto props, que é passado automaticamente para o componente.

Componente de Função

function Saudacao(props) {
  return <h1>Olá, {props.nome}</h1>;
}

Componente de Classe

import React, { Component } from 'react';

class Saudacao extends Component {
  render() {
    return <h1>Olá, {this.props.nome}</h1>;
  }
}
